按钮说明:
=========

第一行 (Y=20, 高度20px):
- 模式切换: 关闭/抽→罐/罐→人 三种模式循环
- 保留: 0-100级可调，鼠标悬停滚轮调整
- 修ON/OFF: 经验修补开关

第二行 (Y=45, 高度20px):
- 全取: 将所有储罐经验取出到玩家
- 取 1 级: 从储罐取出指定等级（滚轮调整）
- 存 1 级: 向储罐存入指定等级（滚轮调整）
- 全存: 将玩家多余经验全部存入储罐

RGB效果说明:
============

1. 标题 "经验泵控制器" (最炫酷):
   - 2秒快速周期
   - 双重色彩叠加 (基础彩虹 + 正弦波偏移)
   - 饱和度1.0, 明度1.0

2. 信息行 "总容/玩家/储罐数" (中等):
   - 5秒较慢周期
   - 三行相位偏移 (0.0, 0.33, 0.66)
   - 饱和度0.9, 明度1.0

3. "无储罐" 警告 (红色闪烁):
   - 1秒快速周期
   - 色相0.0-0.1 (红色范围)
   - 明度0.6-1.0波动

布局计算:
=========

第一行按钮:
- 总宽度: 176 - 8*2 = 160px
- 间距: 5px × 2 = 10px
- 按钮宽度: (160 - 10) / 3 = 50px

第二行按钮:
- 总宽度: 176 - 8*2 = 160px
- 间距: 5px × 3 = 15px
- 按钮宽度: (160 - 15) / 4 = 36.25px ≈ 37px

进度条:

- 宽度: 160px, 高度: 15px
- 填充色: #7EFF05 (绿色)
- 背景色: #555555 (深灰)
