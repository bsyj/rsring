# Tooltip 提示信息改进

## 修改内容

### 1. 经验泵控制器 (ItemExperiencePumpController)

**修改前**：
- 基本信息总是显示：
  - "经验泵控制器"
  - "配合经验储罐使用"
  - "放在背包即可生效"

**修改后**：
- 基本信息只显示：
  - "经验泵控制器"
- Shift 详细信息中显示：
  - "配合经验储罐使用"
  - "放在背包即可生效"
  - 其他功能介绍和使用方法

**改进原因**：
- 减少普通模式下的信息量，使 tooltip 更简洁
- 将详细说明移到 Shift 模式，保持信息层次清晰

### 2. 箱子戒指 (ItemAbsorbRing)

**修改前**：
- 过滤模式显示：
  - "过滤模式: 白名单/黑名单"

**修改后**：
- 过滤模式显示：
  - "过滤模式: 白名单/黑名单"
  - "放在绑定下"（仅当戒指已绑定时显示）

**改进原因**：
- 提示用户戒指需要放在绑定的箱子下方才能工作
- 只在已绑定时显示，避免未绑定时的混淆

## 修改文件

1. `rsring/src/main/java/com/moremod/item/ItemExperiencePumpController.java`
   - 移除普通模式下的"配合经验储罐使用"和"放在背包即可生效"
   - 将这些信息移到 Shift 详细信息的开头

2. `rsring/src/main/java/com/moremod/item/ItemAbsorbRing.java`
   - 在"过滤模式"下方添加"放在绑定下"提示
   - 仅在 `cap.isBound()` 为 true 时显示

## 用户体验改进

1. **信息层次更清晰**：
   - 普通模式显示最重要的状态信息
   - Shift 模式显示详细的功能说明

2. **减少视觉噪音**：
   - 控制器的基本 tooltip 更简洁
   - 只在需要时显示详细信息

3. **上下文相关提示**：
   - 箱子戒指的"放在绑定下"只在已绑定时显示
   - 避免未绑定时的无效提示
